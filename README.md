# 灵影短剧系统 v1.0

> AI漫剧智能创作系统 - 端到端小说改编AI漫剧自动化创作平台

## 📖 项目简介

**灵影短剧系统**是一个专注于小说改编AI漫剧的端到端自动化创作系统。用户只需提供小说内容，系统即可自动生成可直接剪辑成片的配音和视频素材，无需专业剪辑技能。

### 🔍 系统价值

- **🎯 降低创作门槛**：无需专业编剧、导演、剪辑技能
- **⏱️ 提高创作效率**：30章小说 → 1集短剧，全自动流程仅需数小时
- **📈 保证内容质量**：AI驱动的多阶段质量控制流程
- **🎨 风格多样**：支持古风、现代、科幻、武侠等多种风格
- **💰 降低制作成本**：减少人力和时间成本投入

## ✨ 核心功能

- 自动读取、解析、提炼小说内容

## 🎨 支持风格

### 默认风格（default）
适用于各种类型的短剧，通用风格

### 古风风格（ancient）
- **适用场景**：古风、武侠、仙侠类短剧
- **语言风格**：半文半白，古韵盎然
- **视觉风格**：水墨风格，传统美学

### 现代风格（modern）
- **适用场景**：现代都市、职场、校园类短剧
- **语言风格**：口语化，贴近生活
- **视觉风格**：扁平化，色彩鲜明

### 科幻风格（scifi）
- **适用场景**：科幻、赛博朋克、未来世界类短剧
- **语言风格**：科技感强，术语准确
- **视觉风格**：赛博朋克，霓虹灯光

### 武侠风格（wuxia）
- **适用场景**：武侠、江湖类短剧
- **语言风格**：江湖气息，武侠韵味
- **视觉风格**：水墨武侠，气势磅礴

## 🚀 快速开始

### 📋 系统要求

- **操作系统**：Windows 10+ / macOS 10.15+ / Linux
- **Python**：3.8+
- **内存**：8GB+ RAM
- **存储空间**：50GB+ 可用空间
- **网络**：稳定的互联网连接（用于API调用）

### 📦 安装依赖

```bash
pip install -r requirements.txt
```

### 📚 准备小说

1. **拆分小说**（如果是整本书）：
   ```bash
   cd novel_splitter
   python cli.py --input "../novels/《小说名》.txt"
   ```

2. **小说目录结构**：
   ```
   novels/
   ├── 《小说名》.txt          # 完整小说文件（用于拆分）
   └── 小说名/
       └── chapters/
           └── 第一部/
               ├── 第一章-标题.txt
               ├── 第二章-标题.txt
               └── 第三章-标题.txt
   ```

### 🎯 运行系统

#### 读取小说章节
```bash
python cli.py read-chapters --novel "novels/小说名/chapters/第一部" --start 1 --end 3
```

#### 生成剧情梗概
```bash
python cli.py plot-summary --novel "novels/小说名/chapters/第一部" --start 1 --end 3
```

#### 提取核心元素
```bash
python cli.py core-elements --novel "novels/小说名/chapters/第一部" --start 1 --end 3
```

## 📁 输出结果

系统生成的素材将保存在 `output/` 目录中：

- **output/plot_summaries/**：剧情梗概文件
- **output/core_elements/**：核心元素提取结果
- **output/scripts/**：生成的剧本文件
- **output/materials/**：音频和视频素材
- **output/final/**：最终可剪辑成片的素材包

## 🎨 使用技巧

### 🔧 风格选择
- 根据小说类型选择合适的风格
- 古风小说 → 古风风格
- 现代小说 → 现代风格
- 科幻小说 → 科幻风格
- 武侠小说 → 武侠风格

### 📖 章节选择
- 建议每批处理3-5章，保证内容连贯性
- 选择包含完整情节的章节范围
- 优先处理有重要角色出场的章节

### ⚙️ 质量控制
- 检查生成的剧情梗概是否符合原著
- 确认角色信息是否准确完整
- 预览音频和视频素材质量
